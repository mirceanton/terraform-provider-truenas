# truenas_cron_job (Resource)

Manages cron jobs for scheduled task execution.

## Example Usage

### Basic Cron Job

```terraform
# Create a cron job that runs daily at midnight
resource "truenas_cron_job" "cleanup" {
  user        = "root"
  command     = "/usr/local/bin/cleanup.sh"
  description = "Daily cleanup task"
  enabled     = true

  schedule {
    minute = "0"
    hour   = "0"
  }
}
```

### Cron Job with Output Capture

```terraform
resource "truenas_cron_job" "backup_report" {
  user        = "root"
  command     = "/usr/local/bin/backup-report.sh"
  description = "Generate daily backup report"

  schedule {
    minute = "0"
    hour   = "8"
  }

  capture_stdout = true
  capture_stderr = true
}
```

> **Note:** When `capture_stdout` or `capture_stderr` is `true`, output is mailed to the user account.

## Import

Cron jobs can be imported using the numeric ID:

```shell
terraform import truenas_cron_job.example 1
```
