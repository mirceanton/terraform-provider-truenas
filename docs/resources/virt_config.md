# truenas_virt_config (Resource)

Manages the global virtualization configuration on TrueNAS.

-> Requires TrueNAS 25.0 or later.

## Example Usage

```terraform
# Configure the virtualization subsystem
resource "truenas_virt_config" "main" {
  pool       = "tank"
  v4_network = "10.0.0.0/24"
}
```

## Import

The virtualization config is a singleton and can be imported using "virt_config":

```shell
terraform import truenas_virt_config.example virt_config
```
