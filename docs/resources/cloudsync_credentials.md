# truenas_cloudsync_credentials (Resource)

Manages cloud sync credentials for backup tasks.

## Example Usage

### AWS S3 Credentials

```terraform
# Create AWS S3 cloud sync credentials
resource "truenas_cloudsync_credentials" "s3" {
  name = "aws-backup"

  s3 {
    access_key_id     = var.aws_access_key
    secret_access_key = var.aws_secret_key
    region            = "us-west-2"
  }
}
```

### Backblaze B2 Credentials

```terraform
resource "truenas_cloudsync_credentials" "b2" {
  name = "b2-backup"

  b2 {
    account = var.b2_account_id
    key     = var.b2_application_key
  }
}
```

### S3-Compatible Storage (MinIO, Wasabi, etc.)

```terraform
resource "truenas_cloudsync_credentials" "minio" {
  name = "minio-local"

  s3 {
    access_key_id     = var.minio_access_key
    secret_access_key = var.minio_secret_key
    endpoint          = "https://minio.local:9000"
    region            = "us-east-1"
  }
}
```

## Import

Credentials can be imported using the numeric ID:

```shell
terraform import truenas_cloudsync_credentials.example 1
```
